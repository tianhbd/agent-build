# Changelog

All notable changes to this template are documented in this file.

The format is inspired by Keep a Changelog and semantic versioning.

## [0.1.0] - 2026-04-07

### Added

- Initial `agent-build` template structure.
- Task-first governance files: `CLAUDE.md`, `TASKS.md`, `MEMORY.md`.
- Role prompts for `main-agent`, `planner`, `architect`, `coder`, `debugger`, `reviewer`.
- Agent capability definitions in `agents/`.
- Reusable skill playbooks in `skills/`.
- Product, architecture, and decision documentation in `docs/`.
- API, data model, and acceptance specs in `specs/`.
- Delivery workflows in `workflows/`.
